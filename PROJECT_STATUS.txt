4) Как запустить проект локально
- Установить зависимости (если ещё не установлено):
```
npm install
```
- Запустить dev-сервер:
```
npm run dev
```
- Собрать production-версию (опционально):
```
npm run build
```
- Запустить линтер и форматтер:
```
npm run lint
npm run format
```
- Проверить TypeScript без эмита:
```
npx tsc --noEmit
```
